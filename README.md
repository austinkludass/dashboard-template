# Dashboard Template Project
This project contains the scaffold for a dashboard web application.
- Front End - `React` + `Material UI` + `Toastify`
- Back End - Firebase `Auth`, `Firestore`, `Storage`, `Functions`

## How to set up this project
- Clone the repository
- Open the project in VS Code
- Open a terminal at the project path and run `npm install`
- Create a .env file and set key `REACT_APP_FIREBASE_KEY` to the firebase API key
- Once the node modules have been installed, run the app with `npm start`

## Component list
This template contains many components that can be used through out the application, these include:
- Button
- Calendar
- Form
- Table
- TextBlock

## Page list
This template also includes some template pages that can be set up for specific project needs, these include:
- Login page
- Dashboard page
- Users page
